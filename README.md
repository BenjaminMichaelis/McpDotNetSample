## Template

Create a new app in your current directory by running.

```cli
 dotnet new bmichaelis.quickstart.mcpdotnetsample
```
https://github.com/BenjaminMichaelis/DotnetTemplates?tab=readme-ov-file#installing
https://www.nuget.org/packages/BenjaminMichaelis.Dotnet.Templates

### Parameters

- `--no-sln`: Don't include the solution file.
- `--no-tests`: Don't include the test project.

[Default template options](https://learn.microsoft.com/dotnet/core/tools/dotnet-new#options)
